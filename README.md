# Saveurs Corses

Site vitrine one-page pour Saveurs Corses, vendeur de charcuteries et spécialités corses basé à Venette (60280).

## Stack technique
- Next.js 16
- Tailwind CSS
- Déployé sur Vercel

## Structure du site
1. Accueil
2. À propos
3. Galerie photos
4. Produits
5. Agenda des expositions
6. Contact

## Lancer le projet en local
npm run dev
Ouvrir http://localhost:3000

## Déploiement
Automatique via Vercel à chaque git push sur la branche main.
